Managing PATH in PowerShell

  1. Open PowerShell:

    Press Win + X and select "Windows PowerShell" or "Windows PowerShell (Admin)" from the menu. You can also search for "PowerShell" in the Start menu.

  2. Display the current PATH:

    To view your current PATH, enter the following command in PowerShell:

    $env:Path

  3. Add a new directory to the PATH:

    To add a new directory to the PATH, use the following command, replacing C:\your\new\directory with the actual directory path:

    $env:Path += ";C:\your\new\directory"

    This will only temporarily modify the PATH for the current PowerShell session. To make the change permanent, proceed to the next step.

  4. Save the updated PATH permanently:

    To save the updated PATH permanently, open the "Environment Variables" window:

    • Press Win + X, then click on "System".
    • Click on "Advanced system settings" on the right side of the window.
    • In the "System Properties" window, go to the "Advanced" tab and click on "Environment Variables".

    In the "Environment Variables" window, under the "System variables" section, find the Path variable, select it, and click on "Edit". In the "Edit environment variable" window, click on "New" and add the new directory path. Click "OK" to save the changes and close all the open windows.

  5. Verify the updated PATH:

    Close and reopen PowerShell, then run the following command to check if the updated PATH is applied:

    $env:Path

    The new directory should now be included in the PATH.

  6. Remove a directory from the PATH:

    To remove a directory from the PATH, you'll need to follow the same steps as in step 4 to open the "Environment Variables" window. In the "Edit environment variable" window, select the directory you want to remove and click "Delete". Click "OK" to save the changes and close all the open windows.

    Now your PATH should be updated with the desired changes.
